Spam Email Classifier

Overview
This project classifies emails/messages as Spam or Not Spam (Ham) using:
- Machine Learning (Naive Bayes)
- Deep Learning (DistilBERT)

It also compares the performance of traditional and advanced NLP models.

Objectives
- Build a spam detection system  
- Preprocess and clean text data  
- Apply TF-IDF for feature extraction  
- Train and evaluate models  
- Compare model performance  

Technologies
- Python  
- Pandas, NumPy, Scikit-learn  
- TF-IDF Vectorizer  
- Hugging Face Transformers (DistilBERT)  

Models Used
- Naive Bayes – baseline model  
- DistilBERT – advanced NLP model  

Result
DistilBERT performs better than traditional machine learning models.
